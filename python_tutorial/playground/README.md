Playground (Fresh IDE)
======================

The built-in code editor for the Python Interactive Tutorial.
Press **F2** in the tutorial to open it.

Getting Started
---------------

When you open the playground, you'll see:

- **main.py** -- default file for your project code
- **sandbox-code.py** -- your current sandbox code (auto-synced)
- **projects/** -- phase-specific starter projects (created on first launch)

Theme
-----

TokyoNight is applied automatically. You don't need to configure anything.

Return to Tutorial
------------------

Press **Ctrl+Shift+T** to exit the playground and return to the tutorial.
Any changes you made in the editor will be preserved.

Key Shortcuts
-------------

Open the Cheat Sheet from the tutorial (**Ctrl+Shift+P**) for a full
reference, or press **?** inside the playground.

Cross-Platform
--------------

| Platform   | Works |
|------------|-------|
| Linux      | Yes (bundled) |
| macOS      | Yes (bundled, Intel + Apple Silicon) |
| Windows    | Yes (bundled) |

Starter Projects
----------------

Seven phase-specific project directories are created under
`projects/` the first time you open the playground:

- `phase-1-fundamentals/`
- `phase-2-core-python/`
- `phase-3-oop/`
- `phase-4-intermediate/`
- `phase-5-advanced/`
- `phase-6-engineering/`
- `phase-7-ai/`

Each contains a README and starter `.py` files with TODOs.
