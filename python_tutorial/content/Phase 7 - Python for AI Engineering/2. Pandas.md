# Phase 7: Python for AI Engineering  -  2. Pandas

## Definition

Pandas is a Python library for **data manipulation and analysis**. It introduces two primary data structures:
- **Series**  -  a 1D labeled array (like a column in a spreadsheet).
- **DataFrame**  -  a 2D labeled table with rows and columns (like a spreadsheet or SQL table).

Pandas wraps NumPy arrays with row/column labels (index/columns) and adds hundreds of convenience methods for reading, cleaning, grouping, merging, reshaping, and analyzing data.

## Why it exists

Real-world data is messy and heterogeneous:
- Column names, row labels, and mixed dtypes (strings, numbers, dates) can't be stored in a raw NumPy array.
- Operations like "group by category and compute average", "fill missing values", "join two tables", or "read a CSV" require more than just linear algebra.

Pandas bridges the gap between raw computing (NumPy) and real data wrangling, providing SQL‑like operations inside Python.

## How it works

1. **Import**  -  `import pandas as pd`
2. **Load data**  -  `pd.read_csv()`, `pd.read_excel()`, or create from dict/list: `pd.DataFrame({'col': [1,2,3]})`
3. **Inspect**  -  `.head()`, `.info()`, `.describe()`
4. **Clean**  -  handle nulls, duplicates, wrong types
5. **Transform**  -  filter, group, apply, pivot, merge
6. **Export**  -  `.to_csv()`, `.to_excel()`

## Internal working

- A **Series** is a wrapper around a 1D NumPy array plus an **index** (labels for each element).
- A **DataFrame** is a dict of Series (one per column) sharing a **row index**. Columns can have different dtypes.
- The **index** is a Pandas `Index` object (an immutable array) that supports label‑based alignment and fast lookups.
- **GroupBy** works by splitting the data by key(s), applying a function to each group, then combining results  -  all in compiled code for speed.
- **Vectorized operations** across a column/row still run on the underlying NumPy arrays; Pandas adds alignment on the index before the operation.

## Real-world analogy

A **DataFrame** is like an Excel worksheet with:
- Column headers (column names)
- Row numbers on the left (index  -  can be numeric, dates, or strings)
- The ability to write formulas that apply to whole columns at once
- Pivot tables, filtering, sorting, and vlookup/merge  -  all built‑in

A **Series** is just a single column (including the row labels on the left).

### Import and basic creation

```python
import pandas as pd
import numpy as np

# Series
s = pd.Series([10, 20, 30], index=['a', 'b', 'c'])
print(s)
# a    10
# b    20
# c    30
# dtype: int64

# DataFrame from dictionary
df = pd.DataFrame({
    'Name':    ['Alice', 'Bob', 'Charlie'],
    'Age':     [25, 30, 35],
    'Salary':  [50000, 60000, 70000]
})
print(df)
```

```python
# CSV
df_csv = pd.read_csv('data.csv')
```

### DataFrame inspection

```python
df = pd.DataFrame({
    'Name':  ['Alice', 'Bob', 'Charlie', 'David'],
    'Age':   [25, 30, 35, np.nan],

print(df.head(2))         # first 2 rows
```

### Column / row selection

```python
# Column selection (Series)
print(df['Name'])          # single column
print(df[['Name', 'Age']]) # multiple columns  -  DataFrame

# Row selection by label (.loc)
print(df.loc[0])           # first row (by label)
print(df.loc[0:2])         # rows 0,1,2 (label slicing INCLUSIVE)
print(df.loc[[0, 2]])      # rows 0 and 2

# Row selection by integer position (.iloc)
print(df.iloc[0])          # first row
print(df.iloc[0:2])        # rows 0,1 (exclusive  -  like normal Python)
print(df.iloc[[0, 2]])     # rows 0 and 2

# Both
print(df.loc[0:2, 'Name':'Age'])   # rows 0-2, cols Name-Age
print(df.iloc[0:2, 0:2])           # rows 0-1, cols 0-1
```

### Filtering

```python
# Boolean mask
mask = df['Age'] > 30
print(df[mask])            # rows where Age > 30

# Multiple conditions  -  use &, |, ~ (NOT)
print(df[(df['Age'] > 25) & (df['Salary'] < 75000)])

# .query()  -  string expression (cleaner)
print(df.query('Age > 30 and Salary < 75000'))

# .isin()
print(df[df['Name'].isin(['Alice', 'Charlie'])])

# .between()
print(df[df['Age'].between(25, 35)])
```

### Handling missing data

```python
df = pd.DataFrame({
    'A': [1, 2, np.nan, 4],
    'B': [np.nan, 2, 3, 4],
    'C': [1, 2, 3, 4]
})

print(df.isnull())              # boolean mask of nulls
print(df.isnull().sum())        # count nulls per column

# Drop missing
print(df.dropna())              # drop rows with ANY null
print(df.dropna(axis=1))        # drop columns with ANY null
print(df.dropna(thresh=2))      # keep rows with at least 2 non‑null values

# Fill missing
print(df.fillna(0))             # fill with constant
print(df.fillna(df.mean()))     # fill with column mean
print(df.fillna(method='ffill')) # forward fill
print(df['A'].interpolate())    # linear interpolation
```

### Grouping

```python
df = pd.DataFrame({
    'Department': ['Eng', 'Eng', 'Sales', 'Sales', 'HR'],
    'Name':       ['Alice', 'Bob', 'Charlie', 'David', 'Eve'],
    'Salary':     [90000, 80000, 60000, 65000, 55000]
})

# Group by one column
grouped = df.groupby('Department')
print(grouped['Salary'].mean())   # avg salary per department
print(grouped['Salary'].agg(['mean', 'sum', 'count', 'std']))

# Group by multiple columns
df.groupby(['Department', 'Gender'])['Salary'].mean()

# .agg()  -  multiple aggregations at once
print(df.groupby('Department').agg(
    avg_salary=('Salary', 'mean'),
    max_salary=('Salary', 'max'),
    count=('Name', 'count')
))

# .transform()  -  same shape as original, useful for normalising
df['Salary_norm'] = df.groupby('Department')['Salary'].transform(
    lambda x: (x - x.mean()) / x.std()
)
```

### Merge / Join / Concat

```python
employees = pd.DataFrame({
    'emp_id': [1, 2, 3],
    'name':   ['Alice', 'Bob', 'Charlie'],
    'dept_id': [10, 20, 10]
})

departments = pd.DataFrame({
    'dept_id': [10, 20, 30],
    'dept_name': ['Engineering', 'Sales', 'HR']
})

# Merge (SQL‑like join)
merged = pd.merge(employees, departments, on='dept_id', how='inner')
print(merged)

# Left join
merged_left = pd.merge(employees, departments, on='dept_id', how='left')

# Join on index
emp_idx = employees.set_index('emp_id')
dept_idx = departments.set_index('dept_id')
joined = emp_idx.join(dept_idx)

# Concat (stack rows/columns)
df1 = pd.DataFrame({'A': [1, 2]})
df2 = pd.DataFrame({'A': [3, 4]})
concat_rows = pd.concat([df1, df2], axis=0)   # stack vertically
concat_cols = pd.concat([df1, df2], axis=1)   # side by side
```

### Apply functions

```python
df = pd.DataFrame({'A': [1, 2, 3], 'B': [4, 5, 6]})

# .apply() on a column
df['A_sq'] = df['A'].apply(lambda x: x ** 2)

# .apply() on axis=1 (row-wise)
df['sum'] = df.apply(lambda row: row['A'] + row['B'], axis=1)

# .applymap()  -  element-wise on entire DataFrame (deprecated, use .map() for Series)
print(df.applymap(lambda x: x * 10))  # every element * 10

# .map()  -  value mapping on a Series
df['Category'] = df['A'].map({1: 'low', 2: 'mid', 3: 'high'})
```

### Pivot tables

```python
sales = pd.DataFrame({
    'Region': ['North', 'North', 'South', 'South', 'North'],
    'Product': ['A', 'B', 'A', 'B', 'A'],
    'Revenue': [100, 200, 150, 250, 300]
})

# Pivot table
pivot = pd.pivot_table(
    sales,
    values='Revenue',
    index='Region',
    columns='Product',
    aggfunc='sum',
    fill_value=0
)
print(pivot)

# .melt()  -  unpivot (wide → long)
df_wide = pd.DataFrame({'id': [1,2], 'A': [10,20], 'B': [30,40]})
melted = df_wide.melt(id_vars='id', var_name='Product', value_name='Value')
print(melted)
```

### Data cleaning

```python
df = pd.DataFrame({
    'Name':  ['alice', 'BOB', 'Charlie', 'alice ', 'BOB'],
    'Age':   [25, 30, np.nan, 25, 30.0],
    'Email': ['a@b.com', 'b@c.com', None, 'a@b.com', 'B@C.COM']
})

# Duplicates
print(df.duplicated())                # boolean mask
print(df.duplicated(subset=['Name'])) # check dupes on specific column
df_clean = df.drop_duplicates()       # drop duplicate rows
df_clean = df.drop_duplicates(subset=['Name'], keep='first')

# String accessor (.str)
df['Name'] = df['Name'].str.strip().str.title()  # clean whitespace, capitalise
df['Email'] = df['Email'].str.lower()            # standardise case

# Type conversion
df['Age'] = df['Age'].astype('Int64')  # nullable integer (capital I!)

# Renaming columns
df = df.rename(columns={'Name': 'FullName'})

# Replacing values
df['Email'] = df['Email'].fillna('missing@unknown.com')
```

### Exporting

```python
df.to_csv('output.csv', index=False)
df.to_excel('output.xlsx', sheet_name='Sheet1', index=False)
df.to_json('output.json', orient='records')
```

## Creative mini project: Movie ratings analysis

```python
"""
Movie ratings analysis: group by genre, find avg rating
Dataset: small IMDb-like sample
"""
import pandas as pd
import numpy as np

# Simulated data
np.random.seed(42)
movies = pd.DataFrame({
    'title': [f'Movie {i}' for i in range(100)],
    'genre': np.random.choice(['Action', 'Comedy', 'Drama', 'Horror', 'Sci-Fi'], 100),
    'rating': np.round(np.random.uniform(1, 10, 100), 1),
    'year': np.random.randint(1990, 2024, 100),
    'budget_m': np.random.randint(1, 200, 100)
})

# Inspect
print(movies.head())
print(movies.info())
print(movies.describe())

# Clean
print(f"Missing ratings: {movies['rating'].isnull().sum()}")
print(f"Duplicate rows: {movies.duplicated().sum()}")

# 1. Average rating per genre
genre_stats = movies.groupby('genre')['rating'].agg(['mean', 'count', 'std'])
print("\n=== Genre Stats ===")
print(genre_stats.sort_values('mean', ascending=False))

# 2. Most popular genre by count
print("\n=== Movie Count per Genre ===")
print(movies['genre'].value_counts())

# 3. Top 5 highest rated movies
print("\n=== Top 5 Movies ===")
print(movies.nlargest(5, 'rating')[['title', 'genre', 'rating']])

# 4. Rating distribution (buckets)
movies['rating_category'] = pd.cut(
    movies['rating'],
    bins=[0, 4, 6, 8, 10],
    labels=['Bad', 'Okay', 'Good', 'Great']
)
print("\n=== Rating Distribution ===")
print(movies['rating_category'].value_counts())

# 5. Budget vs rating correlation
correlation = movies['budget_m'].corr(movies['rating'])
print(f"\nBudget-Rating correlation: {correlation:.3f}")

# 6. Yearly average rating trend
yearly = movies.groupby('year')['rating'].mean()
print("\n=== Yearly Avg Rating (last 5 years) ===")
print(yearly.tail())

# 7. Pivot table: avg rating per genre per decade
movies['decade'] = (movies['year'] // 10) * 10
pivot = pd.pivot_table(
    movies,
    values='rating',
    index='genre',
    columns='decade',
    aggfunc='mean',
    fill_value=0
)
print("\n=== Avg Rating by Genre & Decade ===")
print(pivot)

print("\nAnalysis complete!")
```

## Common mistakes

| Mistake | Why it's wrong | Correct |
|---------|----------------|---------|
| `df['col'][df['col'] > 5]` | Chained indexing may trigger `SettingWithCopyWarning` | Use `df.loc[df['col'] > 5, 'col']` |
| Modifying a slice of a DataFrame | May or may not affect original; unpredictable | Use `.copy()` or `.loc` assignment |
| `df.dropna(inplace=True)` | `inplace` is deprecated in many Pandas versions | `df = df.dropna()` |
| Iterating rows with `iterrows()` | Extremely slow for large datasets | Vectorize or use `.apply()` with `axis=1` |
| Forgetting `axis` in `dropna` / `drop` | Drops rows by default but you may want columns | Explicit: `axis=0` (rows) or `axis=1` (cols) |
| Using `pd.merge` without specifying `on` or `how` | Defaults to inner join on common columns  -  may silently drop rows | Always specify `on` and `how` explicitly |
| Not handling dtype after fillna | `fillna(0)` on int column converts to float | Use `df['col'] = df['col'].fillna(0).astype(int)` |
| Confusing `.loc` with `.iloc` | `.loc` uses labels (inclusive), `.iloc` uses positions (exclusive at end) | Match the method to your use case |

## Interview questions + answers

**Q1:** What is the difference between `loc` and `iloc`?  
**A:** `loc` accesses by **label** (row index label, column name). `iloc` accesses by **integer position** (0‑based). With `loc`, slices are inclusive of the end; with `iloc`, they are exclusive.

**Q2:** Explain the difference between `apply`, `map`, and `applymap`.  
**A:** `map`  -  transforms a Series element‑wise (also works for value mapping). `apply`  -  works on a DataFrame axis (row/column) or on a Series. `applymap`  -  element‑wise on an entire DataFrame (now deprecated; use `.map()` on each column instead).

**Q3:** How do you handle missing values in a DataFrame?  
**A:** Detect with `.isnull()` / `.isna()`. Remove with `.dropna()`. Fill with `.fillna()` (constant, forward/backward fill, interpolation, or column mean/median).

**Q4:** What does `pd.merge` do? What are the different types of joins?  
**A:** It combines two DataFrames on a key column (like SQL JOIN). `how='inner'` (intersection), `'left'` (all left keys), `'right'` (all right keys), `'outer'` (union).

**Q5:** What is a Pandas GroupBy object?  
**A:** It splits data into groups by key(s), allows applying a function to each group, and combines results. Available methods: `mean()`, `sum()`, `agg()`, `transform()`, `filter()`.

**Q6:** How do you change the data type of a column?  
**A:** `df['col'] = df['col'].astype('float32')`. For nullable integer: `astype('Int64')` (capital I).

**Q7:** What is the difference between `pd.concat` and `pd.merge`?  
**A:** `concat` stacks DataFrames along an axis (rows or columns)  -  does not align on keys. `merge` joins on columns/indices like SQL JOIN, aligning by key values.

**Q8:** How can you detect and remove duplicate rows?  
**A:** `df.duplicated()` returns a boolean Series; `df.drop_duplicates()` removes them. Use `subset` parameter to check only certain columns.

**Q9:** What is the purpose of the `.str` accessor?  
**A:** Provides vectorized string operations (`.lower()`, `.strip()`, `.contains()`, `.replace()`, etc.) on a Series of strings.

**Q10:** How do you parse dates in `read_csv`?  
**A:** Use `parse_dates=['date_column']` parameter, or later `df['date'] = pd.to_datetime(df['date'])`.

## Knowledge check (5-10 questions with answers)

**Q1:** What is the main difference between a Series and a DataFrame?  
**A:** A Series is 1D labeled array; a DataFrame is a 2D table of Series sharing an index.

**Q2:** Which method gives summary statistics for numeric columns?  
**A:** `df.describe()`

**Q3:** How do you select rows where column 'A' is greater than 10?  
**A:** `df[df['A'] > 10]` or `df.query('A > 10')`

**Q4:** What does `df.dropna()` do by default?  
**A:** Drops any row containing at least one NaN value.

**Q5:** What is the output of `pd.DataFrame({'A': [1,2]}).shape`?  
**A:** `(2, 1)`

**Q6:** Which parameter in `pd.merge` specifies the type of join?  
**A:** `how` (inner, left, right, outer).

**Q7:** What does `df.groupby('col')['val'].mean()` do?  
**A:** Groups by `col` and computes the mean of `val` per group.

**Q8:** What does `pd.cut()` do?  
**A:** Bins numeric values into discrete intervals (e.g., age groups).

**Q9:** True or False: `.loc[0:5]` includes row 5.  
**A:** True  -  `.loc` slicing is inclusive of the end label.

**Q10:** How do you reset the index of a DataFrame?  
**A:** `df.reset_index(drop=True)` (drop=True avoids adding old index as a column).

## Revision notes (bullet points summary)

- **Series**: 1D labeled array; **DataFrame**: 2D labeled table.
- **Creation**: `pd.DataFrame({'col': [...]})`, `pd.read_csv()`, `pd.read_excel()`.
- **Inspection**: `.head()`, `.info()`, `.describe()`, `.shape`, `.columns`, `.dtypes`.
- **Selection**: `.loc` (label), `.iloc` (position), `df['col']`, `df[['col1','col2']]`.
- **Filtering**: boolean mask, `.query()`, `.isin()`, `.between()`.
- **Missing data**: `.isnull()`, `.dropna()`, `.fillna()`, `.interpolate()`.
- **GroupBy**: split → apply → combine; `.agg()`, `.transform()`, `.filter()`.
- **Merge/Join**: `pd.merge` (SQL-like), `df.join` (index-based), `pd.concat` (stacking).
- **Apply**: `.apply()` on Series/axis, `.map()` for value mapping, `.str` for strings.
- **Pivot**: `pd.pivot_table()`, `.melt()` for wide↔long reshape.
- **Cleaning**: `.duplicated()`, `.drop_duplicates()`, `.str.strip()`, `.astype()`.
- **Always prefer vectorized Pandas operations** over row‑wise `iterrows()`.
