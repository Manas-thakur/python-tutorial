## Project 1: Calculator

A command-line calculator that performs arithmetic operations in a REPL (Read-Eval-Print-Loop) interface.

### Concepts Covered
- Functions, conditionals, loops
- Input/output, type conversion
- Error handling with try/except
- Infinite loop with break

### Full Code

```python
# calculator.py
print("=== Python Calculator ===\n")
print("Available operations: +, -, *, /, //, %, **")
print("Type 'quit' to exit.\n")

def get_number(prompt):
    """Safely get a number from user input."""
    while True:
        try:
            return float(input(prompt))
        except ValueError:
            print("Invalid number. Try again.")

def calculate(a, op, b):
    """Perform the requested arithmetic operation."""
    if op == "+":
        return a + b
    elif op == "-":
        return a - b
    elif op == "*":
        return a * b
    elif op == "/":
        if b == 0:
            return "Error: Division by zero"
        return a / b
    elif op == "//":
        if b == 0:
            return "Error: Division by zero"
        return a // b
    elif op == "%":
        if b == 0:
            return "Error: Division by zero"
        return a % b
    elif op == "**":
        return a ** b
    else:
        return f"Unknown operator: {op}"

# REPL Loop
while True:
    expression = input("\nEnter expression (e.g., 5 + 3): ").strip()
    
    if expression.lower() == "quit":
        print("Goodbye!")
        break
    
    # Simple parsing: expects "a op b" format
    parts = expression.split()
    if len(parts) != 3:
        print("Format: number operator number (e.g., 5 + 3)")
        continue
    
    a_str, op, b_str = parts
    a = get_number(f"  {a_str} → ")
    b = get_number(f"  {b_str} → ")
    
    result = calculate(a, op, b)
    print(f"  Result: {a} {op} {b} = {result}")
```

## Project 2: Todo App

A task management application that runs in the terminal and persists tasks to a JSON file.

### Concepts Covered
- Functions, collections (lists, dicts)
- File I/O (JSON)
- CRUD operations (Create, Read, Update, Delete)
- String formatting

### Full Code

```python
# todo_app.py
import json
from datetime import datetime

TODO_FILE = "todo.json"

def load_tasks():
    """Load tasks from JSON file."""
    try:
        with open(TODO_FILE, "r") as f:
            return json.load(f)
    except (FileNotFoundError, json.JSONDecodeError):
        return []

def save_tasks(tasks):
    """Save tasks to JSON file."""
    with open(TODO_FILE, "w") as f:
        json.dump(tasks, f, indent=2)

def add_task():
    """Add a new task."""
    title = input("Task title: ").strip()
    if not title:
        print("Title cannot be empty.\n")
        return
    
    tasks = load_tasks()
    task = {
        "id": len(tasks) + 1 if tasks else 1,
        "title": title,
        "done": False,
        "created": datetime.now().isoformat()
    }
    tasks.append(task)
    save_tasks(tasks)
    print(f"Task '{title}' added!\n")

def list_tasks():
    """Display all tasks."""
    tasks = load_tasks()
    if not tasks:
        print("No tasks. Add one!\n")
        return
    
    print(f"\n{'ID':<4} {'Status':<8} {'Title':<30} {'Created':<20}")
    print("-" * 62)
    for t in tasks:
        status = "✅ Done" if t["done"] else "⬜ Pending"
        created = t["created"][:10]
        print(f"{t['id']:<4} {status:<8} {t['title']:<30} {created:<20}")
    print()

def mark_done():
    """Mark a task as completed."""
    tasks = load_tasks()
    task_id = int(input("Enter task ID to mark done: "))
    for t in tasks:
        if t["id"] == task_id:
            t["done"] = True
            save_tasks(tasks)
            print(f"Task '{t['title']}' marked done!\n")
            return
    print("Task not found.\n")

def delete_task():
    """Delete a task."""
    tasks = load_tasks()
    task_id = int(input("Enter task ID to delete: "))
    for i, t in enumerate(tasks):
        if t["id"] == task_id:
            removed = tasks.pop(i)
            save_tasks(tasks)
            print(f"Task '{removed['title']}' deleted!\n")
            return
    print("Task not found.\n")

def show_stats():
    """Show task completion statistics."""
    tasks = load_tasks()
    total = len(tasks)
    done = sum(1 for t in tasks if t["done"])
    pending = total - done
    print(f"\nTotal tasks: {total}")
    print(f"Completed:   {done}")
    print(f"Pending:     {pending}")
    if total > 0:
        print(f"Progress:    {done/total*100:.1f}%\n")
    else:
        print()

def main():
    print("=== Todo App ===\n")
    
    while True:
        print("1. Add Task")
        print("2. List Tasks")
        print("3. Mark Task Done")
        print("4. Delete Task")
        print("5. Show Stats")
        print("6. Exit")
        
        choice = input("\nChoose (1-6): ")
        
        if choice == "1": add_task()
        elif choice == "2": list_tasks()
        elif choice == "3": mark_done()
        elif choice == "4": delete_task()
        elif choice == "5": show_stats()
        elif choice == "6":
            print("Goodbye!")
            break
        else:
            print("Invalid choice.\n")

if __name__ == "__main__":
    main()
```

## Project 3: Expense Tracker

Track your daily expenses, view totals, and filter by category.

### Concepts Covered
- Dictionaries for data organization
- List comprehensions and filtering
- File I/O with JSON
- String formatting and f-strings
- Functions with returns

### Full Code

```python
# expense_tracker.py
import json
from datetime import datetime

EXPENSES_FILE = "expenses.json"
CATEGORIES = ["Food", "Transport", "Entertainment", "Bills", "Shopping", "Other"]

def load_expenses():
    try:
        with open(EXPENSES_FILE, "r") as f:
            return json.load(f)
    except (FileNotFoundError, json.JSONDecodeError):
        return []

def save_expenses(expenses):
    with open(EXPENSES_FILE, "w") as f:
        json.dump(expenses, f, indent=2)

def add_expense():
    print(f"\nCategories: {', '.join(CATEGORIES)}")
    
    description = input("Description: ").strip()
    if not description:
        print("Description required.\n")
        return
    
    try:
        amount = float(input("Amount ($): "))
        if amount <= 0:
            print("Amount must be positive.\n")
            return
    except ValueError:
        print("Invalid amount.\n")
        return
    
    print("Categories:")
    for i, cat in enumerate(CATEGORIES, 1):
        print(f"  {i}. {cat}")
    
    cat_choice = input("Choose category (1-6): ")
    try:
        category = CATEGORIES[int(cat_choice) - 1]
    except (ValueError, IndexError):
        print("Invalid category.\n")
        return
    
    expense = {
        "id": len(load_expenses()) + 1,
        "description": description,
        "amount": amount,
        "category": category,
        "date": datetime.now().isoformat()[:10]
    }
    
    expenses = load_expenses()
    expenses.append(expense)
    save_expenses(expenses)
    print(f"Added: ${amount:.2f} for {description}\n")

def view_all():
    expenses = load_expenses()
    if not expenses:
        print("No expenses.\n")
        return
    
    total = sum(e["amount"] for e in expenses)
    print(f"\n{'Date':<12} {'Description':<20} {'Category':<15} {'Amount':>8}")
    print("-" * 57)
    for e in expenses:
        print(f"{e['date']:<12} {e['description']:<20} {e['category']:<15} ${e['amount']:>6.2f}")
    print("-" * 57)
    print(f"{'':<12} {'':<20} {'TOTAL':<15} ${total:>6.2f}\n")

def view_by_category():
    expenses = load_expenses()
    if not expenses:
        print("No expenses.\n")
        return
    
    print()
    for cat in CATEGORIES:
        cat_expenses = [e for e in expenses if e["category"] == cat]
        if cat_expenses:
            cat_total = sum(e["amount"] for e in cat_expenses)
            print(f"{cat}: {len(cat_expenses)} expense(s) — ${cat_total:.2f}")
            for e in cat_expenses:
                print(f"    {e['date']} — {e['description']}: ${e['amount']:.2f}")
    print()

def monthly_summary():
    expenses = load_expenses()
    if not expenses:
        print("No expenses.\n")
        return
    
    monthly = {}
    for e in expenses:
        month = e["date"][:7]  # YYYY-MM
        monthly[month] = monthly.get(month, 0) + e["amount"]
    
    print("\nMonthly Summary:")
    for month in sorted(monthly.keys()):
        print(f"  {month}: ${monthly[month]:.2f}")
    print()

def main():
    print("=== Expense Tracker ===\n")
    
    while True:
        print("1. Add Expense")
        print("2. View All")
        print("3. View by Category")
        print("4. Monthly Summary")
        print("5. Exit")
        
        choice = input("\nChoose (1-5): ")
        
        if choice == "1": add_expense()
        elif choice == "2": view_all()
        elif choice == "3": view_by_category()
        elif choice == "4": monthly_summary()
        elif choice == "5":
            print("Goodbye!")
            break
        else:
            print("Invalid choice.\n")

if __name__ == "__main__":
    main()
```

## Project 4: Password Generator

Generate secure, random passwords with configurable length and character types.

### Concepts Covered
- Functions with parameters
- `random` module
- String constants
- List comprehensions
- User input validation

### Full Code

```python
# password_generator.py
import random
import string

print("=== Password Generator ===\n")

def get_yes_no(prompt):
    """Get a yes/no answer from the user."""
    while True:
        response = input(prompt).strip().lower()
        if response in ("y", "yes"):
            return True
        elif response in ("n", "no"):
            return False
        print("Please answer y or n.")

def generate_password(length=12, use_uppercase=True, use_lowercase=True,
                      use_digits=True, use_symbols=True):
    """Generate a random password with specified criteria."""
    if not any([use_uppercase, use_lowercase, use_digits, use_symbols]):
        return "Error: At least one character type must be selected."
    
    if length < 4:
        return "Error: Minimum length is 4."
    
    # Build character pool
    chars = ""
    if use_uppercase:
        chars += string.ascii_uppercase
    if use_lowercase:
        chars += string.ascii_lowercase
    if use_digits:
        chars += string.digits
    if use_symbols:
        chars += string.punctuation
    
    # Guarantee at least one character from each selected type
    password = []
    if use_uppercase:
        password.append(random.choice(string.ascii_uppercase))
    if use_lowercase:
        password.append(random.choice(string.ascii_lowercase))
    if use_digits:
        password.append(random.choice(string.digits))
    if use_symbols:
        password.append(random.choice(string.punctuation))
    
    # Fill the rest randomly
    password.extend(random.choice(chars) for _ in range(length - len(password)))
    
    # Shuffle to avoid predictable pattern
    random.shuffle(password)
    
    return "".join(password)

def password_strength(password):
    """Evaluate password strength on a scale of 1-5."""
    score = 0
    
    if len(password) >= 8: score += 1
    if len(password) >= 12: score += 1
    if len(password) >= 16: score += 1
    if any(c.isupper() for c in password): score += 1
    if any(c.islower() for c in password): score += 1
    if any(c.isdigit() for c in password): score += 1
    if any(c in string.punctuation for c in password): score += 1
    
    if score <= 2: return "Very Weak"
    elif score <= 3: return "Weak"
    elif score <= 4: return "Moderate"
    elif score <= 5: return "Strong"
    else: return "Very Strong"

def bulk_generate(count, **kwargs):
    """Generate multiple passwords at once."""
    passwords = []
    for _ in range(count):
        passwords.append(generate_password(**kwargs))
    return passwords

def main():
    print("Let's configure your password:\n")
    
    try:
        length = int(input("Password length (default 16): ") or "16")
    except ValueError:
        length = 16
    
    use_upper = get_yes_no("Include uppercase letters? (Y/n): ")
    use_lower = get_yes_no("Include lowercase letters? (Y/n): ")
    use_digits = get_yes_no("Include digits? (Y/n): ")
    use_symbols = get_yes_no("Include symbols? (Y/n): ")
    
    count = 1
    if get_yes_no("Generate multiple passwords? (y/N): "):
        try:
            count = int(input("How many? "))
        except ValueError:
            count = 5
    
    print(f"\nGenerating {count} password(s)...\n")
    
    passwords = bulk_generate(
        count,
        length=length,
        use_uppercase=use_upper,
        use_lowercase=use_lower,
        use_digits=use_digits,
        use_symbols=use_symbols
    )
    
    print(f"{'Password':<30} {'Strength':<15} {'Length':<8}")
    print("-" * 53)
    for pwd in passwords:
        strength = password_strength(pwd)
        print(f"{pwd:<30} {strength:<15} {len(pwd):<8}")
    
    # Save to file option
    if get_yes_no("\nSave to file? (y/N): "):
        filename = input("Filename (default: passwords.txt): ") or "passwords.txt"
        with open(filename, "w") as f:
            f.write(f"Generated: {datetime.now().isoformat()}\n")
            f.write("=" * 40 + "\n\n")
            for i, pwd in enumerate(passwords, 1):
                f.write(f"{i}. {pwd}  (Strength: {password_strength(pwd)})\n")
        print(f"Saved to {filename}.")

if __name__ == "__main__":
    from datetime import datetime
    main()
```

## Common Mistakes Across Projects

| Mistake | Description | Fix |
|---------|-------------|-----|
| Not handling file errors | App crashes if JSON file is corrupt | Use try/except with defaults |
| Infinite loops | Missing `break` condition in REPL | Always have an exit option |
| Not validating input | App crashes on non-numeric input | Use try/except and re-prompt |
| Hardcoding values | Magic numbers and strings throughout | Use constants and config |
| Ignoring edge cases | Division by zero, empty lists | Check before operating |
| Mixing concerns | Business logic mixed with I/O | Separate into functions |
| No error recovery | App terminates on any error | Use try/except to stay running |

## Interview Questions + Answers

**Q1: What is a REPL loop?**
A1: Read-Eval-Print-Loop — a program that repeatedly reads input, evaluates it, prints the result, and loops. The Calculator and Todo app both use REPL patterns.

**Q2: How do you persist data between program runs?**
A2: Write data to files (JSON, CSV, text). Read the file on startup and save changes before exiting.

**Q3: Why separate the program into functions?**
A3: Functions improve readability, reusability, testability, and maintainability. Each function should do one thing well.

**Q4: How do you handle invalid user input gracefully?**
A4: Use try/except for type errors, loops to re-prompt, and validate ranges/choices before processing.

**Q5: What is the `if __name__ == "__main__":` pattern?**
A5: Ensures code only runs when the script is executed directly, not when imported as a module.

**Q6: How do you generate random numbers in Python?**
A6: `import random` — `random.randint(a,b)` for integers, `random.choice(seq)` for picking from a sequence, `random.shuffle(list)` to shuffle in place.

**Q7: What's the best way to store configuration data?**
A7: JSON files are ideal — human-readable, cross-platform, and natively supported by Python.

**Q8: How do you format currency in Python?**
A8: `print(f"${amount:.2f}")` formats with 2 decimal places and a dollar sign.

**Q9: What is CRUD?**
A9: Create, Read, Update, Delete — the four basic operations for persistent storage. The Todo app is a CRUD application.

**Q10: How do you ensure unique IDs for items?**
A10: Auto-increment: use `max(item["id"] for item in items) + 1` or use `uuid` module.
