# 2. Modules

## Definition
A module in Python is simply a `.py` file containing Python definitions, functions, classes, and variables. Modules allow you to organize code into separate files and reuse them across multiple programs. You import a module's contents using the `import` statement.

## Why it exists
As programs grow, putting everything in one file becomes unmanageable. Modules solve this by:
- **Organization**: Split code into logical files (e.g., `math_utils.py`, `file_ops.py`, `database.py`)
- **Reusability**: Write once, import anywhere. No copy-pasting code
- **Namespace management**: Each module has its own namespace, preventing naming conflicts
- **Maintainability**: Change code in one module and all imports get the update
- **Third-party ecosystem**: Modules are the foundation of Python's rich package ecosystem (PyPI has over 500,000 packages)

## How it works
When you `import` a module, Python:
1. Searches for the module file in `sys.path` (list of directories)
2. If found, executes the module's code from top to bottom
3. Creates a module object with its own namespace
4. Binds the module name (or imported names) in the current namespace

```python
# Different import styles
import math                   # Access: math.sqrt(16)
from math import sqrt         # Access: sqrt(16) directly
from math import sqrt as s    # Access: s(16) with alias
from math import *            # Import ALL names (avoid this!)
```

## Internal working

### Module loading process
1. **Find**: Python looks for the module in `sys.path` — the current directory first, then `PYTHONPATH` environment variable, then standard library paths, then site-packages for third-party modules.
2. **Compile**: If the `.py` file is newer than the corresponding `.pyc` file (in `__pycache__`), Python compiles it to bytecode for faster loading next time.
3. **Execute**: Python runs the module's code in its own namespace (a dictionary). Functions and classes are defined, variables are assigned.
4. **Cache**: The module object is stored in `sys.modules` dictionary. Subsequent imports of the same module just retrieve it from `sys.modules` — the code runs only ONCE per interpreter session.

```python
# What happens internally:
import sys
print(sys.modules['math'])  # Already cached after first import
```

### `__name__ == "__main__"`
When a module is run directly (e.g., `python mymodule.py`), Python sets `__name__` to `"__main__"`. When imported, `__name__` is set to the module's name. This lets a file act both as a reusable module and a standalone script.

```python
# Inside mymodule.py
def greet(name):
    return f"Hello, {name}!"

if __name__ == "__main__":
    # This only runs when executed directly, not when imported
    print(greet("World"))
```

## Real-world analogy
A **toolbox in a garage**:
- Each module is a drawer in the toolbox: one for wrenches (string functions), one for screwdrivers (math functions), one for measuring tape (file operations)
- `import wrench_drawer` — you open the wrench drawer and take what you need
- `from wrench_drawer import crescent_wrench` — you grab just the crescent wrench
- `import wrench_drawer as wd` — you label the drawer "wd" for quicker access
- The `__name__ == "__main__"` is like a label saying "I'm the main workbench" — only when you're working at this bench do you run the demo/test on the tool

### Reloading modules
```python

```

### `__pycache__` directory
After importing a module, Python creates a `__pycache__` folder next to your `.py` files containing compiled `.pyc` files. These are bytecode-compiled versions that load faster on subsequent runs. Python automatically recompiles when the `.py` file changes (based on timestamp).

```
project/
├── __pycache__/
│   └── math_utils.cpython-311.pyc
├── math_utils.py
└── main.py
```

### The `if __name__ == "__main__"` pattern
```python
# --- calculator.py ---
def add(a, b): return a + b
def sub(a, b): return a - b
def mul(a, b): return a * b
def div(a, b): return a / b

def interactive():
    while True:
        cmd = input("Command (add/sub/mul/div/quit): ").strip()
        if cmd == 'quit':
            break
        a = float(input("a: "))
        b = float(input("b: "))
        ops = {'add': add, 'sub': sub, 'mul': mul, 'div': div}
        if cmd in ops:
            print(f"Result: {ops[cmd](a, b)}")
        else:
            print("Unknown command")

if __name__ == "__main__":
    interactive()
```

## Common mistakes

1. **Circular imports** — Module A imports Module B which imports Module A. Causes `ImportError` or `AttributeError`. Restructure to avoid this.
   ```python
   # a.py: from b import func_b  →  b.py: from a import func_a  →  ERROR
   ```

2. **Using `from module import *`** — pollutes the namespace, makes code unclear, can shadow existing names.

3. **Naming conflicts** — your module name shadows a built-in. Don't name files `math.py`, `sys.py`, `os.py`, etc.

4. **Module not found despite being present** — module's directory isn't in `sys.path`. Either install the module properly or adjust the path.

5. **Forgetting `__name__ == "__main__"`** — test/demo code runs when the module is imported, causing side effects.

6. **Importing inside functions for performance** — unnecessary; Python caches modules after first import, so import at top is fine.

7. **Modifying imported module attributes** — changes affect all other importers of the same module since modules are shared singletons.

8. **Case sensitivity** — Python is case-sensitive. `import Math` ≠ `import math`.

9. **Believing each import reloads the module** — modules are cached in `sys.modules`. Second import just retrieves from cache.

10. **Not distinguishing between executable scripts and importable modules** — use the `if __name__` idiom consistently.

## Interview questions + answers

**Q1: What's the difference between `import module` and `from module import name`?**
A: `import module` loads the entire module and requires prefix (`module.func()`). `from module import name` loads the module and binds `name` directly, so you call `func()` without prefix. The latter pollutes the namespace but is more concise for frequently used names.

**Q2: How does Python find modules?**
A: It searches `sys.path`, which contains: the current directory, `PYTHONPATH` env variable directories, standard library paths, and site-packages for third-party modules.

**Q3: What is `__pycache__`?**
A: A directory containing bytecode-compiled `.pyc` files. Python compiles `.py` to bytecode for faster loading. Files are only recompiled if the `.py` source is newer.

**Q4: Explain `if __name__ == "__main__":`.**
A: `__name__` is a special variable Python sets to `"__main__"` when the file is run directly, and to the module name when imported. This idiom makes a file usable both as a script and an importable module without running test/demo code on import.

**Q5: What is `sys.modules`?**
A: A dictionary mapping module names to loaded module objects. Python checks here first before actually importing. All modules imported during a session are cached here.

**Q6: How do you reload an already-imported module?**
A: Use `importlib.reload(module)`. Useful in interactive development when module code changes.

**Q7: What's the purpose of `__all__` in a module?**
A: `__all__` defines the list of names exported when `from module import *` is used. If not defined, all names not starting with `_` are exported.

**Q8: What are `sys.path` and `PYTHONPATH`?**
A: `sys.path` is a Python list of directories searched for imports. `PYTHONPATH` is an environment variable that adds directories to `sys.path`.

## Knowledge check (5-10 questions with answers)

**Q1: If you run `python mymodule.py`, what is `__name__` set to?**
A: `"__main__"`

**Q2: If `mymodule.py` is imported from another file, what is `__name__`?**
A: `"mymodule"` (the module name without `.py`)

**Q3: How many times is a module executed if imported in 3 different files?**
A: Once. After first import, it's cached in `sys.modules`.

**Q4: What directory contains compiled bytecode files?**
A: `__pycache__/`

**Q5: What's wrong with naming your file `random.py`?**
A: It shadows the built-in `random` module. `import random` would load your file, not the standard library.

**Q6: What does `sys.path[0]` typically contain?**
A: The directory of the script being run (or current directory in interactive mode).

**Q7: How do you import a module from a different directory?**
A: Add the directory to `sys.path` (`sys.path.append("/path")`) or use relative imports within packages.

**Q8: What does `from module import *` import?**
A: All names defined in the module that don't start with `_` (underscore), or names listed in `__all__` if defined.

**Q9: Can you import a module without executing it?**
A: No. The import process executes all top-level code in the module. Use `if __name__ == "__main__"` to prevent test code from running.

**Q10: What's the extension of compiled bytecode files?**
A: `.pyc` (e.g., `module.cpython-311.pyc`)

## Revision notes (bullet points summary)

- A module is any `.py` file — functions, classes, variables defined inside it
- `import module` → access with `module.name`; `from module import name` → direct access
- `as` creates an alias: `import numpy as np`
- Modules are cached in `sys.modules` — code runs only once per session
- `sys.path` controls where Python looks for modules (current dir, standard lib, site-packages)
- `if __name__ == "__main__"` lets a file be both script and module
- `__pycache__` stores `.pyc` bytecode for faster loading
- `importlib.reload(module)` forces re-execution
- Avoid circular imports, naming conflicts with built-ins, and `from module import *`
- `__all__` controls what's exported with `*`
- `PYTHONPATH` environment variable adds directories to module search path

### Creative Mini Project: Create a `math_utils` Module with Reusable Functions

```python
# --- math_utils.py ---
"""
A collection of reusable math utility functions.
Supports basic operations, statistics, geometry, and number theory.
"""

import math
import random

# ============================================================
# BASIC OPERATIONS
# ============================================================

def add(a, b):
    return a + b

def subtract(a, b):
    return a - b

def multiply(a, b):
    return a * b

def divide(a, b):
    if b == 0:
        raise ZeroDivisionError("Cannot divide by zero")
    return a / b

def power(base, exp):
    return base ** exp

def modulus(a, b):
    if b == 0:
        raise ZeroDivisionError("Cannot mod by zero")
    return a % b

def floor_divide(a, b):
    if b == 0:
        raise ZeroDivisionError("Cannot divide by zero")
    return a // b

# ============================================================
# STATISTICS
# ============================================================

def mean(numbers):
    if not numbers:
        raise ValueError("List is empty")
    return sum(numbers) / len(numbers)

def median(numbers):
    if not numbers:
        raise ValueError("List is empty")
    sorted_nums = sorted(numbers)
    n = len(sorted_nums)
    mid = n // 2
    if n % 2 == 0:
        return (sorted_nums[mid - 1] + sorted_nums[mid]) / 2
    return sorted_nums[mid]

def mode(numbers):
    if not numbers:
        raise ValueError("List is empty")
    freq = {}
    for n in numbers:
        freq[n] = freq.get(n, 0) + 1
    max_freq = max(freq.values())
    modes = [k for k, v in freq.items() if v == max_freq]
    return modes[0] if len(modes) == 1 else modes  # return first or all

def standard_deviation(numbers):
    if len(numbers) < 2:
        raise ValueError("Need at least 2 values")
    m = mean(numbers)
    variance = sum((x - m) ** 2 for x in numbers) / (len(numbers) - 1)
    return math.sqrt(variance)

# ============================================================
# GEOMETRY
# ============================================================

PI = math.pi

def circle_area(radius):
    if radius < 0:
        raise ValueError("Radius cannot be negative")
    return PI * radius ** 2

def circle_circumference(radius):
    if radius < 0:
        raise ValueError("Radius cannot be negative")
    return 2 * PI * radius

def rectangle_area(length, width):
    if length < 0 or width < 0:
        raise ValueError("Dimensions cannot be negative")
    return length * width

def triangle_area(base, height):
    if base < 0 or height < 0:
        raise ValueError("Dimensions cannot be negative")
    return 0.5 * base * height

def sphere_volume(radius):
    if radius < 0:
        raise ValueError("Radius cannot be negative")
    return (4 / 3) * PI * radius ** 3

def pythagorean(a, b):
    """Return hypotenuse of right triangle."""
    if a < 0 or b < 0:
        raise ValueError("Sides cannot be negative")
    return math.sqrt(a ** 2 + b ** 2)

# ============================================================
# NUMBER THEORY
# ============================================================

def is_even(n):
    return n % 2 == 0

def is_odd(n):
    return n % 2 != 0

def factorial(n):
    if n < 0:
        raise ValueError("Factorial not defined for negative numbers")
    if n == 0:
        return 1
    return n * factorial(n - 1)

def is_prime(n):
    if n < 2:
        return False
    for i in range(2, int(math.sqrt(n)) + 1):
        if n % i == 0:
            return False
    return True

def gcd(a, b):
    """Greatest Common Divisor using Euclidean algorithm."""
    while b:
        a, b = b, a % b
    return abs(a)

def lcm(a, b):
    """Least Common Multiple."""
    if a == 0 or b == 0:
        return 0
    return abs(a * b) // gcd(a, b)

def prime_factors(n):
    """Return list of prime factors."""
    factors = []
    d = 2
    while d * d <= n:
        while n % d == 0:
            factors.append(d)
            n //= d
        d += 1
    if n > 1:
        factors.append(n)
    return factors

# ============================================================
# UTILITY
# ============================================================

def clamp(value, min_val, max_val):
    """Clamp value between min and max."""
    return max(min_val, min(value, max_val))

def lerp(a, b, t):
    """Linear interpolation between a and b at t (0-1)."""
    if t < 0 or t > 1:
        raise ValueError("t must be between 0 and 1")
    return a + (b - a) * t

def map_range(value, in_min, in_max, out_min, out_max):
    """Map value from one range to another."""
    if in_min == in_max:
        raise ValueError("Input range cannot be zero")
    return out_min + (value - in_min) * (out_max - out_min) / (in_max - in_min)

# ============================================================
# DEMO / TEST (only runs when executed directly)
# ============================================================

if __name__ == "__main__":
    print("=" * 50)
    print("  MATH UTILS MODULE — SELF TEST")
    print("=" * 50)

    print("\n--- Basic Operations ---")
    print(f"  add(10, 5)          = {add(10, 5)}")
    print(f"  subtract(10, 5)     = {subtract(10, 5)}")
    print(f"  multiply(3, 7)      = {multiply(3, 7)}")
    print(f"  divide(10, 3)       = {divide(10, 3):.4f}")
    print(f"  power(2, 10)        = {power(2, 10)}")
    print(f"  modulus(17, 5)      = {modulus(17, 5)}")

    print("\n--- Statistics ---")
    nums = [4, 8, 6, 5, 3, 4, 5, 5]
    print(f"  Data: {nums}")
    print(f"  mean                = {mean(nums)}")
    print(f"  median              = {median(nums)}")
    print(f"  mode                = {mode(nums)}")
    print(f"  std_dev             = {standard_deviation(nums):.4f}")

    print("\n--- Geometry ---")
    print(f"  circle_area(5)       = {circle_area(5):.4f}")
    print(f"  circle_circum(5)    = {circle_circumference(5):.4f}")
    print(f"  rect_area(4, 6)     = {rectangle_area(4, 6)}")
    print(f"  triangle_area(4, 6) = {triangle_area(4, 6)}")
    print(f"  sphere_volume(5)    = {sphere_volume(5):.4f}")
    print(f"  pythagorean(3, 4)   = {pythagorean(3, 4)}")

    print("\n--- Number Theory ---")
    print(f"  is_even(7)          = {is_even(7)}")
    print(f"  is_odd(7)           = {is_odd(7)}")
    print(f"  factorial(7)        = {factorial(7)}")
    print(f"  is_prime(29)        = {is_prime(29)}")
    print(f"  is_prime(1)         = {is_prime(1)}")
    print(f"  gcd(48, 18)         = {gcd(48, 18)}")
    print(f"  lcm(12, 18)         = {lcm(12, 18)}")
    print(f"  prime_factors(84)   = {prime_factors(84)}")

    print("\n--- Utility ---")
    print(f"  clamp(15, 0, 10)    = {clamp(15, 0, 10)}")
    print(f"  lerp(0, 100, 0.5)   = {lerp(0, 100, 0.5)}")
    print(f"  map_range(5, 0, 10, 0, 100) = {map_range(5, 0, 10, 0, 100)}")

    print("\n" + "=" * 50)
    print("  All tests passed!")
    print("=" * 50)
```
