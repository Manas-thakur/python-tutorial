# 6. Pathlib

## Definition
`pathlib` is a Python module that provides an object-oriented interface for working with filesystem paths. Instead of representing paths as strings (like `os.path` does), `pathlib` uses `Path` objects with methods and properties for all common file operations. It was introduced in Python 3.4 and is the modern, recommended way to handle filesystem paths.

## Why it exists
- **Cross-platform**: Handles Windows backslashes and Unix forward slashes automatically  -  write code once, works everywhere
- **Readable**: `data / "subdir" / "file.txt"` is clearer than `os.path.join("data", "subdir", "file.txt")`
- **Object-oriented**: Paths have methods like `.exists()`, `.is_file()`, `.read_text()`  -  no need for separate `os.path.exists(path)` calls
- **Composable**: Operations chain naturally: `Path("data").glob("*.txt").read_text()`
- **Modern replacement**: Cleaner than `os.path.join()`, `os.listdir()`, `glob.glob()`, `shutil` for many common tasks
- **Type safety**: A `Path` object is clearly a path, not a string that could be anything

## How it works

### Creating Path objects
```python
from pathlib import Path

# Current directory
p = Path()
p = Path(".")

# Relative path
p = Path("data/subdir/file.txt")

# Absolute path
p = Path("/home/user/projects/data/file.txt")

# Home directory
p = Path.home()

# Current working directory
p = Path.cwd()

# Combine with /
script_dir = Path(__file__).parent
log_dir = script_dir / "logs"
log_file = log_dir / "app.log"
```

### Path properties
```python
p = Path("/home/user/projects/data/config.json")

print(p.name)        # config.json
print(p.stem)        # config
print(p.suffix)      # .json
print(p.parent)      # /home/user/projects/data
print(p.parents[0])  # /home/user/projects/data (same as .parent)
print(p.parents[1])  # /home/user/projects
print(p.parents[2])  # /home/user
print(p.anchor)      # /  (the root directory)
print(p.parts)       # ('/', 'home', 'user', 'projects', 'data', 'config.json')
```

### Path methods
```python
p = Path("test.txt")

# Check existence and type
p.exists()      # True/False
p.is_file()
p.is_dir()
p.is_symlink()
p.is_absolute()

# Create/delete directories
Path("new_dir").mkdir()                            # Single directory
Path("a/b/c").mkdir(parents=True)                  # Create parent directories too
Path("new_dir").mkdir(exist_ok=True)               # Don't error if exists
Path("old_dir").rmdir()                            # Remove empty directory

# Delete files
Path("temp.txt").unlink()                          # Delete file
Path("temp.txt").unlink(missing_ok=True)           # Don't error if missing (3.8+)

# Rename/move
Path("old.txt").rename("new.txt")
Path("old.txt").replace("existing.txt")            # Replace even if exists
```

### Reading and writing files
```python
# Read entire file as text
content = Path("file.txt").read_text(encoding="utf-8")

# Read entire file as bytes
data = Path("image.jpg").read_bytes()

# Write text (creates file, overwrites existing)
Path("output.txt").write_text("Hello, World!", encoding="utf-8")

# Write bytes
Path("output.bin").write_bytes(b"binary data")

# WARNING: These load the entire file into memory. For large files, use open().
```

### Globbing (finding files)
```python
# Find all .py files in current directory (non-recursive)
for py_file in Path().glob("*.py"):
    print(py_file)

# Find all .txt files recursively in all subdirectories
for txt_file in Path().rglob("*.txt"):
    print(txt_file)

# rglob with pattern
for file in Path("src").rglob("**/*.py"):
    print(file)

# List all files and directories
for item in Path(".").iterdir():
    if item.is_file():
        print(f"FILE: {item.name}")
    elif item.is_dir():
        print(f"DIR:  {item.name}")
```

## Internal working

### Path is a system of classes
`pathlib` uses a class hierarchy to separate POSIX (Unix/Linux/macOS) and Windows path semantics:

```
Path (concrete factory)
├── PurePath (pure computation, no I/O)
│   ├── PurePosixPath    # Unix-style paths
│   └── PureWindowsPath  # Windows-style paths
└── Path (with I/O operations)
    ├── PosixPath        # Unix with I/O
    └── WindowsPath      # Windows with I/O
```

- **PurePath subclasses**: Operate on path strings without touching the filesystem (no `.exists()`, `.is_file()`, etc.)
- **Path subclasses**: Full filesystem access  -  calling `Path()` on Linux returns `PosixPath`; on Windows returns `WindowsPath`

### Slash operator
The `/` operator is overloaded via `__truediv__`. It joins path components:

```python
base = Path("/home/user")
full = base / "projects" / "python" / "main.py"
```

This internally calls `PurePath.__truediv__`, which joins with the appropriate separator (`/` or `\`).

### Path resolution
`Path.resolve()` resolves symbolic links and returns the absolute path:

```python
# /home/user is a symlink to /data/user
p = Path("/home/user/file.txt")
print(p.resolve())  # /data/user/file.txt
```

### Backward compatibility
You can convert between `Path` objects and strings freely:
```python
p = Path("file.txt")
str(p)              # "file.txt"
open(p, "r")        # Path works with open()
os.path.exists(p)   # Path works with os.path functions too
```

## Real-world analogy
A **GPS navigation system**:
- `Path("home/documents/report.pdf")` is like setting a destination address
- `.parent` is going back one step ("go to the district level")
- `.name` is the building name ("report.pdf")
- `.stem` is the building without the extension ("report")
- `.suffix` is the building type identifier (".pdf")
- `.exists()` checks if the address actually exists on the map
- `.glob("*.pdf")` is like searching all buildings of type PDF on the current street
- `.rglob("*.pdf")` is searching the entire city for PDFs
- The `/` operator is like saying "take this route, then turn here, then stop at this building"

## Common mistakes

1. **Using `os.path` and `pathlib` inconsistently**  -  Pick `pathlib` and stick with it. It works with `open()`, `os.path` functions, etc.

2. **Forgetting `parents=True` on `mkdir`**  -  `Path("a/b/c").mkdir()` fails if `a/b` doesn't exist. Add `parents=True`.

3. **Not handling `FileExistsError`**  -  `mkdir()` fails if directory exists. Use `exist_ok=True`.

4. **Assuming `read_text()`/`write_text()` work for large files**  -  They load the entire file into memory. For large files, use `open()` with iteration.

5. **Path concatenation with `+`**  -  `Path("dir") + "file.txt"` raises `TypeError`. Use `/`: `Path("dir") / "file.txt"`.

6. **Forgetting that `.iterdir()` is not recursive**  -  It only lists the immediate directory. Use `.rglob("*")` for recursive.

7. **Checking `==` on paths**  -  `Path("dir/")` and `Path("dir")` might not be equal even though they refer to the same location. Use `.resolve()` or `.samefile()`.

8. **Mixing `Path` and string when concatenating**  -  `Path("a") / "b" / "c"` is fine. But `Path("a") + "/b"` is wrong.

9. **Not using `with_name()` and `with_suffix()`**  -  Manually replacing extensions with string manipulation is error-prone. Use `.with_suffix('.csv')`.

10. **Windows path issues**  -  If you hardcode `/` separators on Windows, they work (pathlib handles it). But using raw Windows paths with `\` needs escape: `Path("C:\\Users")`.

## Interview questions + answers

**Q1: What's the advantage of `pathlib` over `os.path`?**
A: `pathlib` provides an object-oriented interface with readable operators (`/`), cross-platform handling, and method chaining. It's more intuitive and reduces boilerplate.

**Q2: How do you build a path from components?**
A: Use the `/` operator: `Path("base") / "subdir" / "file.txt"`.

**Q3: How do you find all `.txt` files recursively?**
A: `Path().rglob("*.txt")` or `Path().glob("**/*.txt")`.

**Q4: What's the difference between `.glob()` and `.rglob()`?**
A: `.glob()` matches in the current directory only (non-recursive). `.rglob()` matches recursively in all subdirectories.

**Q5: What's the difference between `.name`, `.stem`, and `.suffix`?**
A: `.name` = full filename (`image.png`). `.stem` = filename without suffix (`image`). `.suffix` = extension (`.png`).

**Q6: How do you create nested directories?**
A: `Path("a/b/c").mkdir(parents=True, exist_ok=True)`.

**Q7: Can you use `Path` with the built-in `open()`?**
A: Yes, `Path` objects are compatible with `open()`, `os.path` functions, and any function that accepts a path string.

**Q8: What does `Path(__file__).parent` do?**
A: Gets the directory containing the current Python script. Useful for building paths relative to the script location.

**Q9: How do you change a file's extension?**
A: `p.with_suffix('.json')` returns a new path with the extension changed.

**Q10: What's `Path.home()`?**
A: Returns a Path object pointing to the user's home directory (`~`).

### Creative Mini Project: File System Explorer Tool

```python
"""
File System Explorer Tool
A CLI tool for exploring, analyzing, and organizing files using pathlib.
"""

from pathlib import Path
import datetime
import shutil
import sys


class FileExplorer:
    """Interactive file system explorer with analysis and organization features."""

    def __init__(self, root="."):
        self.root = Path(root).resolve()
        if not self.root.exists():
            raise FileNotFoundError(f"Directory not found: {root}")
        if not self.root.is_dir():
            raise NotADirectoryError(f"Not a directory: {root}")

    def list_contents(self, path=None, show_hidden=False, pattern=None):
        """List contents of a directory."""
        path = Path(path) if path else self.root
        if not path.exists():
            print(f" Path not found: {path}")
            return []

        entries = []
        for item in path.iterdir():
            if not show_hidden and item.name.startswith("."):
                continue
            if pattern and not item.match(pattern):
                continue
            entries.append(item)

        # Sort: directories first, then files, alphabetically
        entries.sort(key=lambda x: (not x.is_dir(), x.name.lower()))

        print(f"\n {path} ({len(entries)} items)")
        print("─" * 60)

        dirs = [e for e in entries if e.is_dir()]
        files = [e for e in entries if e.is_file()]

        for d in dirs:
            print(f"   {d.name}/")

        for f in files:
            size = f.stat().st_size
            size_str = format_size(size)
            mtime = datetime.datetime.fromtimestamp(f.stat().st_mtime)
            print(f"   {f.name:<35} {size_str:>8}  {mtime:%Y-%m-%d %H:%M}")

        return entries

    def tree(self, path=None, prefix="", max_depth=3, current_depth=0):
        """Display directory tree."""
        if current_depth > max_depth:
            return
        path = Path(path) if path else self.root
        try:
            entries = sorted(
                path.iterdir(),
                key=lambda x: (not x.is_dir(), x.name.lower())
            )
        except PermissionError:
            print(f"{prefix}   Permission denied")
            return

        for i, entry in enumerate(entries):
            is_last = i == len(entries) - 1
            connector = "└── " if is_last else "├── "
            if entry.is_dir():
                print(f"{prefix}{connector} {entry.name}/")
                extension = "    " if is_last else "│   "
                self.tree(entry, prefix + extension, max_depth, current_depth + 1)
            else:
                size = format_size(entry.stat().st_size)
                print(f"{prefix}{connector} {entry.name} ({size})")

    def search(self, pattern, path=None):
        """Search for files matching a glob pattern recursively."""
        path = Path(path) if path else self.root
        results = list(path.rglob(pattern))
        print(f"\n Searching for '{pattern}' in {path}")
        print(f"   Found {len(results)} results:")
        for r in sorted(results):
            rel = r.relative_to(self.root)
            size = format_size(r.stat().st_size)
            print(f"    {rel} ({size})")
        return results

    def analyze(self, path=None):
        """Analyze directory structure and sizes."""
        path = Path(path) if path else self.root
        print(f"\n Analysis: {path}")
        print("═" * 50)

        total_files = 0
        total_dirs = 0
        total_size = 0
        extensions = {}
        largest_files = []
        oldest_files = []
        newest_files = []

        for item in path.rglob("*"):
            try:
                if item.is_file():
                    total_files += 1
                    stat = item.stat()
                    total_size += stat.st_size

                    ext = item.suffix.lower() or "(no ext)"
                    extensions[ext] = extensions.get(ext, 0) + 1

                    largest_files.append((item, stat.st_size))
                    oldest_files.append((item, stat.st_mtime))
                    newest_files.append((item, stat.st_mtime))

                elif item.is_dir():
                    total_dirs += 1
            except (PermissionError, OSError):
                continue

        # Sort lists
        largest_files.sort(key=lambda x: x[1], reverse=True)
        oldest_files.sort(key=lambda x: x[1])
        newest_files.sort(key=lambda x: x[1], reverse=True)

        print(f"  Total directories: {total_dirs:,}")
        print(f"  Total files:       {total_files:,}")
        print(f"  Total size:        {format_size(total_size)}")
        print(f"  Average file size: {format_size(total_size // max(total_files, 1))}")

        print(f"\n  Top 5 file extensions:")
        for ext, count in sorted(extensions.items(), key=lambda x: x[1], reverse=True)[:5]:
            print(f"    {ext:<12} {count} files")

        print(f"\n  Top 5 largest files:")
        for item, size in largest_files[:5]:
            rel = item.relative_to(self.root)
            print(f"    {format_size(size):>8}  {rel}")

        dt_now = datetime.datetime.now()
        print(f"\n  Oldest files:")
        for item, mtime in oldest_files[:3]:
            rel = item.relative_to(self.root)
            dt = datetime.datetime.fromtimestamp(mtime)
            days_ago = (dt_now - dt).days
            print(f"    {dt:%Y-%m-%d} ({days_ago}d ago)  {rel}")

        print(f"\n  Newest files:")
        for item, mtime in newest_files[:3]:
            rel = item.relative_to(self.root)
            dt = datetime.datetime.fromtimestamp(mtime)
            days_ago = (dt_now - dt).days
            print(f"    {dt:%Y-%m-%d} ({days_ago}d ago)  {rel}")

    def organize_by_extension(self, target_dir=None, dry_run=True):
        """Organize files by extension into subdirectories."""
        target = Path(target_dir) if target_dir else self.root
        if not target.exists():
            print(f" Directory not found: {target}")
            return

        files = [f for f in target.iterdir() if f.is_file() and not f.name.startswith(".")]

        if dry_run:
            print(f"\n Dry run: Preview of organizing {len(files)} files in {target}")
        else:
            print(f"\n Organizing {len(files)} files in {target}")

        print("═" * 60)

        organized = 0
        for f in files:
            ext = f.suffix[1:] if f.suffix else "no_extension"
            ext_dir = target / ext.upper()
            dest = ext_dir / f.name

            if dry_run:
                print(f"  Would move: {f.name} → {ext.upper()}/{f.name}")
            else:
                ext_dir.mkdir(exist_ok=True)
                shutil.move(str(f), str(dest))
                print(f"  Moved: {f.name} → {ext.upper()}/{f.name}")
            organized += 1

        if dry_run:
            print(f"\n  Run with dry_run=False to actually move files.")
        else:
            print(f"\n   Organized {organized} files.")

    def file_info(self, path):
        """Show detailed info about a file or directory."""
        p = Path(path).resolve()
        if not p.exists():
            print(f" Not found: {p}")
            return

        print(f"\n  Path: {p}")
        print(f"  Name: {p.name}")
        print(f"  Stem: {p.stem}")
        print(f"  Suffix: {p.suffix}")
        print(f"  Parent: {p.parent}")
        print(f"  Home: {Path.home() in p.parents}")

        if p.is_file():
            stat = p.stat()
            print(f"  Type: File")
            print(f"  Size: {format_size(stat.st_size)}")
            print(f"  Created:  {datetime.datetime.fromtimestamp(stat.st_ctime)}")
            print(f"  Modified: {datetime.datetime.fromtimestamp(stat.st_mtime)}")
            print(f"  Accessed: {datetime.datetime.fromtimestamp(stat.st_atime)}")
            print(f"  Permissions: {oct(stat.st_mode)[-3:]}")
        elif p.is_dir():
            files = sum(1 for _ in p.rglob("*") if _.is_file())
            dirs = sum(1 for _ in p.rglob("*") if _.is_dir())
            print(f"  Type: Directory")
            print(f"  Total files: {files}")
            print(f"  Total dirs:  {dirs}")


def format_size(size_bytes):
    """Format file size in human-readable format."""
    for unit in ['B', 'KB', 'MB', 'GB', 'TB']:
        if size_bytes < 1024:
            return f"{size_bytes:.1f} {unit}" if unit != 'B' else f"{size_bytes} B"
        size_bytes /= 1024
    return f"{size_bytes:.1f} PB"


def interactive_explorer():
    """Run an interactive file explorer session."""
    start = input("Enter directory to explore (press Enter for current): ").strip() or "."
    explorer = FileExplorer(start)

    print(f"\n File Explorer  -  {explorer.root}")
    print("Commands: list, tree, search <pattern>, analyze, organize, info <path>, quit")
    print("─" * 60)

    current_path = explorer.root

    while True:
        try:
            cmd = input(f"\n{current_path.name} > ").strip().split(maxsplit=1)
            if not cmd:
                continue

            action = cmd[0].lower()
            arg = cmd[1] if len(cmd) > 1 else ""

            if action in ("q", "quit", "exit"):
                print("Goodbye!")
                break

            elif action == "list":
                explorer.list_contents(current_path)

            elif action == "tree":
                depth = int(arg) if arg.isdigit() else 3
                explorer.tree(current_path, max_depth=depth)

            elif action == "search":
                if arg:
                    explorer.search(arg, current_path)
                else:
                    print("Usage: search <pattern>")

            elif action == "analyze":
                explorer.analyze(current_path)

            elif action == "organize":
                explorer.organize_by_extension(current_path, dry_run=True)
                confirm = input("Execute organization? (yes/no): ")
                if confirm.lower() == "yes":
                    explorer.organize_by_extension(current_path, dry_run=False)

            elif action == "info":
                if arg:
                    explorer.file_info(current_path / arg)
                else:
                    print("Usage: info <filename>")

            elif action == "cd":
                if arg:
                    new_path = (current_path / arg).resolve()
                    if new_path.exists() and new_path.is_dir():
                        current_path = new_path
                        print(f"Changed to: {current_path}")
                    else:
                        print(f" Not a valid directory: {new_path}")
                else:
                    print("Usage: cd <dirname>")

            else:
                print("Unknown command. Try: list, tree, search, analyze, organize, info, cd, quit")

        except KeyboardInterrupt:
            print("\nGoodbye!")
            break
        except Exception as e:
            print(f" Error: {e}")


if __name__ == "__main__":
    interactive_explorer()
```
