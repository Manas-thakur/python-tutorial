# Codédex Monthly Challenge Submission

## Title

Learn Python & Build 7 Real Projects — All from Your Terminal

## Body

Most coding tutorials teach you syntax, then expect you to figure out how to build real projects on your own. Others throw you straight into a project without building the fundamentals first.

This one bridges both gaps — and does it entirely in your terminal.

**The terminal is the point.** No browser tabs, no notifications, no context-switching. Just you and a focused programming environment that takes you from your first `print()` to building a text editor, a chat server, an API framework, and more.

**How it works:**

1. **Learn the fundamentals** — 56 topics across 7 phases covering Python basics through OOP, concurrency, and AI engineering. Each topic is displayed one digestible section at a time with syntax-highlighted code.

2. **Test your knowledge** — 150+ coding challenges with auto-validation, quizzes after every topic, and spaced-repetition flashcards. Earn XP, level up, and unlock the next phase at 70% mastery.

3. **Experiment freely** — Press F2 to open a built-in code playground (Fresh IDE) with syntax highlighting, file tree, and integrated terminal. Write, run, and debug code without ever leaving the terminal.

4. **Build real projects** — 7 step-by-step guided walkthroughs that put everything together. Each project has 8 steps, a starter skeleton, and a complete solution:

   - **Expense Tracker** — SQLite database, CLI with argparse, CSV import/export, ASCII bar charts
   - **TUI Text Editor** — Gap buffer data structure, raw terminal mode, ANSI rendering, regex tokenizer
   - **Markdown Blog Engine** — Markdown→HTML parser, template engine with inheritance, SSE live reload, RSS feed
   - **Chat Server & Client** — asyncio TCP server, JSON protocol, chat rooms, rate limiting
   - **Pixel Art Editor** — Sparse 2D grid, Bresenham line drawing, flood fill, Sobel filters, 24-bit ANSI color
   - **Task Scheduler** — Cron/interval triggers, subprocess executor, web dashboard, decorator API
   - **API Framework** — URL routing with path params, middleware pipeline, HTTP parsing, decorator-based routes

Every project is built from scratch using only Python's standard library. Zero external dependencies.

**What you'll walk away with:**
- Solid Python fundamentals that apply everywhere
- 7 working projects you can show off
- The muscle memory of building real software from scratch
- A terminal workflow that keeps you focused

Install in one command:

```bash
pip install git+https://github.com/Manas-thakur/python-tutorial.git
pytut
```
