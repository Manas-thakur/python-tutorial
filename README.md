# Python Interactive Tutorial

Learn Python from zero to building 7 real projects — all from your terminal, zero distractions.

```bash
pip install git+https://github.com/Manas-thakur/python-tutorial.git
pytut
```

## Why the Terminal?

No browser tabs. No notifications. No context-switching. Just you, Python, and a focused coding environment that teaches you fundamentals, tests your knowledge with challenges, lets you experiment in a sandbox, and guides you through building real projects — all without leaving the terminal.

## The Learning Path

```
Learn ───► Practice ───► Experiment ───► Build
  │            │              │              │
  │  56 topics  │  150+ challenges │  Code sandbox  │  7 projects
  │  7 phases   │  Quizzes        │  Fresh IDE     │  8 steps each
  │  AI to engineering│ Flashcards    │                │
```

### 1. Learn — 56 Topics Across 7 Phases

| Phase | Topics |
|-------|--------|
| Phase 1: Python Fundamentals | 11 topics — variables, loops, functions |
| Phase 2: Core Python | 8 topics — modules, packages, file I/O, pathlib |
| Phase 3: Object-Oriented Programming | 8 topics — classes, inheritance, magic methods |
| Phase 4: Intermediate Python | 9 topics — decorators, generators, context managers |
| Phase 5: Advanced Python | 7 topics — metaclasses, descriptors, concurrency |
| Phase 6: Python for Engineering | 6 topics — NumPy, Pandas, Matplotlib |
| Phase 7: Python for AI Engineering | 7 topics — PyTorch, LangChain, RAG, agents |

Each topic is displayed one section at a time with syntax-highlighted code. No overwhelming walls of text — just focused, digestible lessons.

### 2. Practice — 150+ Challenges + Quizzes + Flashcards

Every topic ends with a coding challenge. Type real code, get it validated against expected output. Difficulty ramps from easy to medium to hard. Track your XP, level up, earn badges.

### 3. Experiment — Built-in Code Sandbox / Fresh IDE

Press `F2` to open a full terminal-based IDE where you can write, run, and debug Python code. Comes with syntax highlighting, a file tree, and integrated terminal. All your sandbox code auto-saves between sessions.

### 4. Build — 7 Project Tutorials

After learning the fundamentals, build real projects with step-by-step guided walkthroughs. Each project includes a starter skeleton and a complete solution.

| Project | Difficulty | What You'll Learn |
|---------|-----------|-------------------|
| **Expense Tracker** | Beginner | SQLite, argparse, CSV parsing, ASCII charts |
| **TUI Text Editor** | Intermediate | Gap buffers, raw terminal mode, ANSI codes, regex tokenization |
| **Markdown Blog Engine** | Intermediate | Markdown parsing, template engine, SSE live reload, RSS |
| **Chat Server & Client** | Intermediate | asyncio, TCP protocols, room management, auth |
| **Pixel Art Editor** | Intermediate | 2D arrays, flood fill, Bresenham lines, image filters |
| **Task Scheduler** | Advanced | Cron triggers, decorators, subprocess execution, web dashboard |
| **API Framework** | Advanced | URL routing, middleware, HTTP parsing, decorator-based APIs |

## Key Features

- **Interactive reading** — sections one at a time, syntax-highlighted code
- **Knowledge checks** — Q&A review after each topic
- **Coding challenges** — 150+ exercises with expected-output validation
- **Flashcards** — spaced-repetition review with self-rating
- **Focused learning** — XP, levels, streaks, badges, progressive unlocking
- **Bookmark & resume** — auto-saves your spot, prompts to continue
- **Error explainer** — plain-English explanations for 13 common Python errors
- **Full-text search** — search across all topics instantly
- **Cross-platform** — works on Linux, macOS, Windows

## Quick Start

```bash
# Install
pip install git+https://github.com/Manas-thakur/python-tutorial.git

# Launch
pytut
```

### Commands

| Command | Description |
|---------|-------------|
| `python-tutorial` | Interactive menu (default) |
| `list` | List all phases and topics |
| `view <phase> <topic>` | View a specific topic |
| `quiz [phase]` | Take a quiz |
| `challenge <phase> <topic>` | Do a coding challenge |
| `projects` | Browse project tutorials |
| `sandbox` | Open the code playground |
| `search <term>` | Search all content |
| `flashcards [phase]` | Review as flashcards |
| `status` | Show progress, level, badges |
| `bookmark` | Show your current spot |

## Keyboard Shortcuts (TUI)

| Key | Action |
|-----|--------|
| `F2` | Open Fresh IDE playground |
| `F3` | Browse project tutorials |
| `Ctrl+F` | Search topics |
| `Ctrl+Q` | Start quiz |
| `Ctrl+Shift+F` | Flashcards |
| `Ctrl+B` | Toggle sidebar |
| `Up/Down` | Previous/Next topic |
| `Left/Right` | Previous/Next section |
